READLIST

// things worth reading

A living index of links I've found useful, interesting, or just want to remember. Skewed heavily towards systems programming, hardware documentation, and whatever's been making the rounds on Hacker News.

// ARCHITECTURE & HARDWARE

  • Intel® 64 and IA-32 Architectures Software Developer Manuals

    The full x86/x86-64 ISA reference. Volumes 1–4. Essential if you're doing anything at the instruction level, writing a JIT, or just want to understand what the CPU is actually doing.

    x86 ISA Intel
  • AMD64 Architecture Programmer's Manual

    AMD's counterpart to the Intel SDM. Good to read both when you care about cross-vendor compatibility, particularly around NUMA topology and MSRs.

    x86-64 AMD ISA
  • Linux NUMA Memory Policy — kernel.org

    kernel.org documentation on NUMA memory policies. Useful companion when you're optimising for locality and need to understand how the kernel exposes the topology.

    NUMA Linux Memory

// LINUX & KERNEL

  • Linux Driver Implementer's API Guide

    The most useful single-entry documentation point for kernel driver development. Covers device model, DMA, IRQs, bus interfaces, and more.

    Linux Kernel Drivers
  • LWN.net

    The best ongoing coverage of Linux kernel development, systems programming, and open source projects. The weekly kernel development summaries alone are worth bookmarking.

    Linux Kernel News

// NETWORKING & TELECOMS

  • 3GPP Specifications (by series)

    The 5G standards starting point. Series 38 covers NR (New Radio); series 23 covers architecture. Dense reading but there's no substitute when you need to know what the standard actually says.

    5G 3GPP Standards
